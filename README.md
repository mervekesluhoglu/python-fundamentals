# Python Fundamentals for Biomedical Engineering 🧬💻

Bu depo, Biyomedikal Mühendisliği çalışmalarımda ve yapay zeka (AI in Health) yolculuğumda kullandığım Python temellerini ve algoritmalarını içermektedir.

## 📂 Depo İçeriği

* **01_Basics:** Değişken tanımlama, veri tipleri ve temel Python sözdizimi.
* **02_Functions:** Geometrik hesaplamalar (alan/çevre) ve modüler kod yapısı.
* **03_Algorithms:** Öklid mesafesi hesaplama ve veri analizi algoritmaları.
